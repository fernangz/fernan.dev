Resources

A curated list of external free resources and applications to help with FrontEnd development. All resources listed are free and open source where possible.

💻 Code Editors

  • 📝
    Visual Studio Code A free, open-source code editor from Microsoft with excellent support for web development, extensions, and integrated terminal.

🎨 Design & Graphics Tools

  • 🖊️
    Inkscape A powerful, free SVG creation application for vector graphics design and illustration.
  • 🎨
    Krita A free digital painting application designed for concept art, illustrations, and matte paintings.
  • 🧊
    Blender A complete 3D creation suite for modeling, animation, rendering, and more. Free and open source.

🎬 Audio & Video Tools

  • 🎞️
    Kdenlive A free and open-source video editor with multi-track editing, effects, and transitions.
  • 🎵
    Audacity A free, open-source audio editing and recording application with powerful features.

🔤 Open Source Fonts

  • 🅰️
    Outfit An open-source sans-serif font, modern and clean. Perfect for UI design.
  • ⌨️
    Fira Code An open-source monospace font with programming ligatures. Great for code editors.
  • 📖
    Rasa An open-source serif font with excellent readability for long-form text.
  • 😊
    Noto Emoji A symbols and emoji font available in monochrome and color variants from Google.

🎯 Icon Collections

  • 🔷
    Material Design Icons A comprehensive collection of icons available in SVG format from Google's Material Design system.
