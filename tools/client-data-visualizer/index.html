About This Data

All data shown above is collected client-side using standard JavaScript APIs. This is the same type of data that websites can access when you visit them.

  • Navigator data: Browser and platform information
  • Screen data: Display properties and capabilities
  • Network data: Connection type and speed estimates
  • Storage data: Cookies and local storage usage
  • Media devices: Available cameras and microphones
  • Location data: IP-based geolocation from Cloudflare
  • Interactions: Real-time mouse, scroll, and keyboard tracking
