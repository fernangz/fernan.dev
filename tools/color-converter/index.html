About Color Formats

  • HEX: Hexadecimal notation (#RRGGBB) used in HTML/CSS
  • RGB: Red, Green, Blue values (0-255) for screen display
  • HSL: Hue, Saturation, Lightness - more intuitive for humans
  • HSB/HSV: Hue, Saturation, Brightness/Value - used in color pickers
