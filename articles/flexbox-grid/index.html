CSS Flexbox & Grid

Master modern CSS layout systems for flexible, responsive designs

Introduction

CSS Flexbox and Grid are powerful layout systems that have revolutionized how we build web layouts. Flexbox excels at one-dimensional layouts (rows OR columns), while Grid handles two-dimensional layouts (rows AND columns). Together, they replace floats and positioning hacks with intuitive, purpose-built tools.

CSS Flexbox

Flexbox is designed for one-dimensional layouts. It's perfect for navigation bars, card layouts, and centering content.

Flexbox Container Properties

.container {
    display: flex;
    flex-direction: row; /* or column */
    justify-content: flex-start; /* main axis alignment */
    align-items: stretch; /* cross axis alignment */
    flex-wrap: wrap; /* allow wrapping */
    gap: 1rem; /* space between items */
}

Flexbox Item Properties

.item {
    flex-grow: 0; /* don't grow */
    flex-shrink: 1; /* shrink if needed */
    flex-basis: auto; /* base size */
    /* Shorthand */
    flex: 0 1 auto;
    /* Or for equal-width items */
    flex: 1;
}

Common Flexbox Patterns

Centering content:

.center {
    display: flex;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
}

Space-between navigation:

.nav {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

Equal-width columns:

.columns {
    display: flex;
    gap: 1rem;
}
.column {
    flex: 1;
}

CSS Grid

Grid is designed for two-dimensional layouts. It excels at page layouts, complex card grids, and any design requiring precise row and column control.

Grid Container Properties

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* 3 equal columns */
    grid-template-rows: auto;
    gap: 1.5rem;
    /* Or use grid-template for both */
    grid-template: 
        "header header header" 80px
        "sidebar main main" 1fr
        "footer footer footer" 60px
        / 200px 1fr 1fr;
}

Grid Item Properties

.item {
    grid-column: 1 / 3; /* span columns 1-3 */
    grid-row: 1 / 2; /* span row 1 */
    /* Or use shorthand */
    grid-area: 1 / 1 / 2 / 3;
    /* Or use named areas */
    grid-area: header;
}

Common Grid Patterns

Responsive card grid:

.grid {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
    gap: 1.5rem;
}

Holy Grail layout:

.layout {
    display: grid;
    grid-template-areas:
        "header header header"
        "sidebar main aside"
        "footer footer footer";
    grid-template-columns: 200px 1fr 200px;
    grid-template-rows: auto 1fr auto;
    min-height: 100vh;
}

Centered single item:

.center {
    display: grid;
    place-items: center;
    min-height: 100vh;
}

Flexbox vs Grid: When to Use Which

Use Flexbox When Use Grid When
One-dimensional layout Two-dimensional layout
Content-driven sizing Precise control over layout
Navigation, cards in a row Page layouts, complex grids
Aligning items in one direction Controlling both rows and columns

Combining Flexbox and Grid

The best layouts often combine both systems:

/* Grid for page layout */
.page {
    display: grid;
    grid-template-areas:
        "header"
        "main"
        "footer";
}

/* Flexbox for component layout */
.header {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

.card-grid {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
    gap: 1.5rem;
}

/* Flexbox inside cards */
.card {
    display: flex;
    flex-direction: column;
}

.card-content {
    flex: 1;
}

Browser Support

Both Flexbox and Grid have excellent browser support:

  • Flexbox: All modern browsers (IE10+)
  • Grid: All modern browsers (IE11+ with prefixes)
  • Use Autoprefixer for legacy browser support

Conclusion

Flexbox and Grid are complementary tools. Flexbox excels at component-level layouts and one-dimensional arrangements, while Grid shines at page-level layouts and complex two-dimensional designs. Master both, and you'll be able to build any layout efficiently.
