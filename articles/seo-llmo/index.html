SEO & LLMO Basics

Optimize for search engines and AI assistants

Introduction

SEO (Search Engine Optimization) helps your content rank in search results. LLMO (Large Language Model Optimization) is the emerging practice of optimizing content for AI assistants like ChatGPT, Claude, and Google's AI Overviews. Both are essential for modern content visibility.

Technical SEO Fundamentals

Meta Tags

<!-- Essential meta tags -->
<title>Page Title - Brand Name</title>
<meta name="description" content="Compelling description under 160 characters">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- Open Graph for social sharing -->
<meta property="og:title" content="Page Title">
<meta property="og:description" content="Description">
<meta property="og:image" content="image.jpg">
<meta property="og:url" content="https://example.com/page">
<meta property="og:type" content="website">

<!-- Twitter Card -->
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Page Title">
<meta name="twitter:description" content="Description">

Canonical URLs

<!-- Prevent duplicate content issues -->
<link rel="canonical" href="https://example.com/original-page">

Structured Data (Schema.org)

<script type="application/ld+json">
{
    "@context": "https://schema.org",
    "@type": "Article",
    "headline": "Article Title",
    "author": {
        "@type": "Person",
        "name": "Author Name"
    },
    "datePublished": "2024-01-01",
    "description": "Article description"
}
</script>

Content Optimization

Heading Hierarchy

<h1>Main Title (one per page)</h1>
<h2>Major Sections</h2>
<h3>Subsections</h3>
<h4>Further divisions</h4>

Keyword Best Practices

  • Include primary keyword in title, H1, and first paragraph
  • Use related keywords naturally throughout
  • Avoid keyword stuffing
  • Focus on user intent, not just keywords

LLMO Optimization

LLMO focuses on making content easily understood and citable by AI models:

Clear, Authoritative Content

  • Use clear, factual language
  • Cite sources and include data
  • Structure content logically
  • Answer questions directly

Q&A Format

<section>
    <h2>What is FrontEnd Development?</h2>
    <p>FrontEnd development is the practice of creating the visual 
    and interactive elements of websites that users interact with 
    directly in their browsers.</p>
</section>

<section>
    <h2>How does the DOM work?</h2>
    <p>The DOM (Document Object Model) is a tree-like representation 
    of your HTML that JavaScript can manipulate to create dynamic 
    content.</p>
</section>

Definition Lists

<dl>
    <dt>DOM</dt>
    <dd>Document Object Model - a programming interface for web documents</dd>
    
    <dt>CSS</dt>
    <dd>Cascading Style Sheets - language for describing presentation</dd>
</dl>

Performance & Core Web Vitals

Google uses Core Web Vitals as ranking factors:

  • LCP (Largest Contentful Paint): < 2.5s
  • FID (First Input Delay): < 100ms
  • CLS (Cumulative Layout Shift): < 0.1

Mobile Optimization

  • Responsive design is mandatory
  • Mobile-first indexing is default
  • Touch-friendly navigation
  • Fast mobile loading

Internal Linking

<!-- Descriptive anchor text -->
<a href="/articles/dom">Learn about the DOM tree</a>

<!-- Avoid generic text -->
<a href="/articles/dom">Click here</a> <!-- BAD -->

Image Optimization

<img 
    src="image.webp"
    alt="Descriptive alt text for accessibility and SEO"
    width="800"
    height="600"
    loading="lazy"
>

URL Structure

  • Use descriptive, readable URLs
  • Include keywords naturally
  • Use hyphens, not underscores
  • Keep URLs short and meaningful
Good: /articles/web-accessibility-guide
Bad: /page.php?id=123&cat=5

SEO Tools

  • Google Search Console: Monitor search performance
  • Google Analytics: Track user behavior
  • Lighthouse: Audit performance and SEO
  • Ahrefs/SEMrush: Keyword research
  • Screaming Frog: Technical SEO audit

LLMO Tools

  • Google AI Test Kitchen: Test AI responses
  • Perplexity: See how AI cites your content
  • AnswerThePublic: Find questions people ask

SEO Checklist

  • ✅ Unique, descriptive title tags
  • ✅ Compelling meta descriptions
  • ✅ Proper heading hierarchy
  • ✅ Mobile-responsive design
  • ✅ Fast page load times
  • ✅ Descriptive alt text for images
  • ✅ Internal linking structure
  • ✅ Schema markup where appropriate
  • ✅ Canonical URLs
  • ✅ HTTPS encryption

Conclusion

SEO and LLMO work together to maximize your content's visibility. Focus on creating high-quality, well-structured content that serves users first. Technical optimization supports great content—it doesn't replace it.
