Browser DevTools Guide

Master the built-in tools for debugging and optimization

Introduction

Browser DevTools are the most important tools in a FrontEnd developer's toolkit. Built into every modern browser, they allow you to inspect, debug, and optimize your websites without leaving the browser. This guide covers essential DevTools features.

Opening DevTools

  • Windows/Linux: F12 or Ctrl+Shift+I
  • Mac: Cmd+Option+I
  • Right-click: Inspect element
  • Menu: More tools → Developer tools

Elements Panel

Inspect and modify HTML and CSS in real-time:

  • Click any element to inspect it
  • Edit HTML by double-clicking
  • Toggle CSS properties on/off
  • Modify values with arrow keys
  • Copy element or styles (right-click)

Pro tip: Use document.querySelector() in Console to find elements.

Console Panel

Execute JavaScript and view logs:

// Log messages
console.log('Info');
console.warn('Warning');
console.error('Error');

// Table for objects
console.table([{name: 'Item', price: 10}]);

// Timing
console.time('operation');
// ... code ...
console.timeEnd('operation');

// Select elements
const el = document.querySelector('.my-class');

// Monitor events
monitorEvents(document.body, 'click');

Sources Panel

Debug JavaScript with breakpoints:

  • Click line numbers to set breakpoints
  • Use debugger statement in code
  • Step through code (F10, F11)
  • Watch variables and expressions
  • View call stack and scope

Network Panel

Monitor all network requests:

  • See all requests (HTML, CSS, JS, images)
  • Check timing and size
  • Filter by type or search
  • Throttle network speed
  • Replay or block requests

Pro tip: Use "Slow 3G" to test mobile performance.

Performance Panel

Profile and optimize performance:

  • Record page load or interaction
  • View flame chart of activities
  • Identify long tasks
  • Find layout thrashing
  • Measure paint and composite time

Application Panel

Manage storage and PWA features:

  • View/edit LocalStorage and SessionStorage
  • Inspect cookies
  • View IndexedDB
  • Check Service Workers
  • Test PWA offline functionality

Lighthouse Panel

Audit for quality and best practices:

  • Performance score
  • Accessibility score
  • Best Practices score
  • SEO score
  • PWA compliance

Run audits for mobile and desktop separately.

Device Mode

Test responsive designs:

  • Toggle with Ctrl+Shift+M or device icon
  • Select preset devices or custom dimensions
  • Test touch events
  • Simulate device pixel ratio
  • Test different network conditions

Color Picker

Built-in color tools:

  • Click any color swatch in Styles
  • Picker with eyedropper
  • Convert between formats (HEX, RGB, HSL)
  • Check contrast ratio

Useful Shortcuts

Shortcut Action
Ctrl/Cmd + P Quick open files
Ctrl/Cmd + Shift + P Run command
Ctrl/Cmd + / Toggle comment
Esc Toggle console drawer

Conclusion

DevTools are essential for modern web development. Master these tools to debug faster, optimize better, and build higher quality websites. Practice using them daily—they'll become second nature.
