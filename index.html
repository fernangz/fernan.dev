Hey there, I'm Fernán García de Zúñiga

Welcome to my personal website. I've been working on the FrontEnd for over two decades now and I wanted to share my thoughts on FrontEnd design and development. Here you'll find tools to make UI creation easier, articles on FrontEnd topics, and useful resources for FrontEnd design and implementation.

As for the articles, if you are new to FrontEnd development, you might be interested in the basics first: The DOM tree, Using CSS to create simple and good user interfaces, Multiples of 8 in FrontEnd Design and ECMAScript, JavaScript or JS. If you're already familiar with these subjects, visit the articles section for more advanced topics.

On the tools section you'll find a variety of utilities from simple copy/paste for symbols, MIME types, ISO codes, HTML entities, to a color palette generator and more.
