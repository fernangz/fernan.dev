Brief

I'm Fernán García de Zúñiga. I work in FrontEnd development, and this site reflects what I've gathered along the way.

The tools section contains utilities I built for my own workflow—color converters, CSS generators, code formatters, and various reference guides. They're all free, run entirely in your browser, and don't require any installation. Use them if they're helpful.

The articles cover topics I've found worth writing about: DOM fundamentals, CSS cascade and specificity, JavaScript events, layout systems, accessibility, security, and performance. They range from introductory material to more advanced subjects. Start wherever makes sense for where you are.

The resources section lists external tools and applications I've found useful over time: code editors, design software, fonts, and icon collections. All are free or open source. These are simply recommendations based on what has worked for me.

Browse around and take what's useful. Leave the rest.
