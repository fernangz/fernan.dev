I'm Fernán García de Zúñiga.

I work in FrontEnd development, and this site is a collection of tools, articles, and resources I've gathered along the way.

The tools are utilities I built for my own workflow: color tools with WCAG contrast checking, box-shadow generator, JSON formatter, HTML entities converter, Base64 encoder/decoder, CSS unit converter, and reference guides for MIME types, ISO language codes, and symbols & emojis. They're all free, run entirely in your browser, and require no installation.

The articles are organized as a structured learning path in three phases: Foundations (how the web works, HTML, CSS, JavaScript basics), Intermediate Skills (ES6+, async programming, closures, prototypes, advanced CSS typography and units), and Professional Skills (SEO & LLMO, web security, accessibility, performance optimization). Start wherever matches your current level.

The resources section curates free and open-source tools across categories: documentation (MDN, W3C, WCAG), development tools (Node.js, Bun, Deno, Docker, Git), code quality (ESLint, Prettier, Jest, Cypress), design software (Blender, Excalidraw, Krita), fonts, icons, online tools, and browser DevTools.

Browse around and take what's useful. Leave the rest.
