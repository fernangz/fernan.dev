Hey there, I'm Fernán García de Zúñiga,

Welcome to my personal website, I've been working on the FrontEnd for over two decades now and I wanted to share my thoughts on FrontEnd design and development, also to include some tools I've cooked up to make UI creation a bit easier, lastly, there are some usefull links to resources for FrontEnd design and implementation.

As for the articles, if you are new into this FrontEnd adventure, you might be interested in the basics first: The DOM tree, Using CSS to create simple and good user interfaces, Multiples of 8 in FrontEnd Design and ECMAScript, JavaScript or JS. If you're already familiar with these subjects and want to check out more advanced topics, please visit the articles section to find something more suitable for you.

On the tools section you'll find a variety of them going from the simple copy/paste for symbols, mime types, ISO codes, HTML entities, etc… to a color palette generator, to some more goodies if you want to check it out.
